# DormManage
 ##  ⏰ 系统介绍
 - 一款简单的宿舍管理系统
 - 基于java语言
 - 无框架构建，Servlet+Jsp
 - 实现了简单的增删改查
 - 支持多用户登录（admin、manager、stu）
